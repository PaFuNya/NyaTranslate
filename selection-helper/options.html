基础设置

控制划词助手的基础行为模式。

行为控制
不在输入框中划词
自动屏蔽 input、textarea、 CodeMirror、Monaco 等文本编辑器区域内的划词操作， 防止干扰正常输入。
触摸模式
为触屏设备优化交互。开启后使用 touchend 事件代替 mouseup， 适合平板等触控场景。

语言匹配

只对包含指定语言的文本触发查词。不勾选任何语言表示不过滤（对所有文本生效）。

语言选择

划选文本包含以下语言时，才会触发查词面板。

严格匹配模式
关闭（默认）：文本中检测到的语种，只要有任意一种在启用列表中，即触发。
开启：文本中所有检测到的语种都必须在启用列表中，才触发（更严格）。

关于

划词助手的版本信息与使用说明。

NyaTransalte
Version 3.0.0
基于 Manifest V3 的 Chrome 扩展，接入 DeepSeek 与通义千问，
让 AI 驱动的翻译与术语解释触手可及。对标沙拉查词交互体验。
使用说明
  1. 在 API 配置 页填入 DeepSeek 或通义千问的 API Key 并保存。
  2. 在 触发规则 页配置各场景的触发方式（默认：划词显示小图标）。
  3. 在任意网页上，用鼠标划选文字（1 ~ 500 字符）。
  4. 点击出现的气泡图标，DeepSeek 和通义千问将同时返回结果。
  5. 点击面板顶部 📌 固定 按钮可钉住面板，方便连续查词。
🔒 隐私声明： 所有 API Key 仅通过 chrome.storage.local 保存在本机， 不会传输至任何第三方服务器。每次查询的文本直接发往你配置的 API 端点。
