关于

划词助手的版本信息与使用说明。

划词助手
Version 2.0.0
基于 Manifest V3 的 Chrome 扩展，接入 DeepSeek 与通义千问，
让 AI 驱动的翻译与术语解释触手可及。
使用说明
  1. 在 API 配置 页面填入 DeepSeek 或通义千问的 API Key 并保存。
  2. 在任意网页上，用鼠标划选文字（1 到 500 个字符）。
  3. 松开鼠标后，光标右下角会出现一个蓝紫色气泡图标。
  4. 点击图标，展开面板，选择 翻译 或 解释术语。
  5. 在 DeepSeek 和通义千问两个 Tab 之间切换，对比结果。
🔒 隐私声明： 所有 API Key 仅通过 chrome.storage.local 保存在本机， 不会传输至任何第三方服务器。每次查询的文本直接发往你配置的 API 端点。
